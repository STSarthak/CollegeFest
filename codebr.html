CODE BREAKER

There will be a two round for this event.

Round-1(Prelims): Time allowed 15 minutes

  • A test will be conducted consist of multiple choice questions based on programming, fundamental and latest IT technology.
  • Test consist of 30 multiple choice questions.
  • Participant will attempt these on the same paper provided by the coordinators.
  • There will be a two test series- A and B
  • Note: Only shortlisted candidates will go for final round.
  • Answer sheets will be evaluated by Faculty coordinators of both campuses. Thereafter result will be conveyed to selected candidates by Student coordinators.

Round 2-(Final): Time allowed 20 minutes.

  • This round is based on programming implemented in computer Lab. Candidates can use C, C++ and Java Language to implement.
  • Test consist of few programs to solve them using above mentioned language.
  • Note: This round will be evaluated by external judges.
