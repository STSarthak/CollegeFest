AIM-2-GAME

Round I

Quiz with Multiple Choice Questions will be handed to the teams and the teams are required to attempt and mark it on the given handout. Maximum 10 teams will be finalized with maximum marks for the Round II.


Round II

 The requirement of the competition is to produce a minimum of 3 pages of website based on the Handout given to them.
Limit: Only one (1) submission per team will be judged. Team size is restricted to be of maximum 2 members.


Standards Compliance:

 Web pages must employ standards-based web design with well formed HTML and CSS. Students can use either MS FrontPage or Notepad Editor for designing the webpages.
  1. Every HTML opening tag must have a matching closing tag. Tags that have no closing tag should include a space and the closing slash at the end of the tag.
  2. Meta tag is a must.
  3. Web pages must be fully functional in Internet Explorer, Firefox and Chrome web browser.
  4. All images should be coded using the ALT attribute.
  5. The first page of the website must index.html.
  6. Every Home page and other pages should have a Horizontal Navigation bar designed in CSS.
  7. Atleast one webpage should be interactive (E.g. Can have Registration Page, Login Page etc.)
Cascading Style Sheets: Cascading Style Sheets must be created from the scratch, for the Navigation Bar.
